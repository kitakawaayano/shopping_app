# shopping_app


## 商品一覧

 * 共通する機能
    * 商品の詳細確認
    * 商品検索
    * 値段順または店名による並び替え・絞り込み

 * 管理者の場合
    * 商品の編集・削除
    * 新規商品の追加

 * 利用者の場合
    * 商品をカートに追加


## 買い物カート
 * 利用者のみ
    * 現在購入中の商品一覧
    * カート中商品の個数、合計金額の表示
    * 商品が**1つ**でもある場合は`購入手続きへ`ボタンが表示される
    * `削除`ボタンでカートから商品を取り消す
    * 同じ商品を別々のタイミングでカートに入れた際は個数が増える


## 購入明細
 * 利用者のみ
    * 今回のカートの合計金額が表示される
    * 支払方法を選択し、`購入`ボタンを押すと購入完了


## 購入履歴
 * 利用者のみ
    * 過去に購入した商品の一覧を表示
